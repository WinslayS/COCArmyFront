Питомцы

  • Король: могучий як / копатик / феникс
  • Королева: единорог
  • Принц: электро сова / феникс
  • Хранитель: единорог / снежок / копатик / медуза
  • Чемпионка: феникс / лиса

Способности

  • Король: ботинки + мяч / ярость + перчатка / мяч + усы
  • Королева: хилки + стрела / хилки + зеркало / в специфических случаях морозная или инвиз
  • Принц: кукольные пресмешники + сфера (штаны не работают как нужно)
  • Хранитель: вечность + ярость или исцеление (наземный микс) / ярость + шар (заход от Хранителя) / кристалл жизни + вечность (драконы, встадники на кабанах, шахтеры)
  • Чемпионка: ракета + электроботинки / ракета + щит / ботинки + спеша
